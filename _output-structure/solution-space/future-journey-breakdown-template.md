## Template Notes for Agent 7

**When filling this template:**
- Be very specific and detailed - this is where you zoom in
- Show step-by-step experience, not just high-level flow
- Explain WHY each change makes experience better
- Connect each improvement to specific solutions/features
- Use concrete examples and scenarios
- Think about actual interface and interactions
- Consider all user types and edge cases
- Include metrics at this detailed level
- Describe what user sees, thinks, feels at each point
- Make it tangible enough for designers to prototype

**Use this breakdown when:**
- A journey stage is particularly complex
- There are significant persona variations
- The experience transformation is critical to value
- Multiple solutions come together in one stage
- You need to detail a specific scenario in depth

**Remember:**
- This supports the consolidated journey, doesn't replace it
- More detail = better for design and development
- Focus on experience, not technical implementation
- Show the transformation clearly: before → after
