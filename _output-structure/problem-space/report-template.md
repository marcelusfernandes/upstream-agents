### Implementation Guidelines

#### Canvas Configuration for Figma Make
**HORIZONTAL LAYOUT SPECIFICATIONS:**
- **Canvas Orientation:** Landscape (horizontal scrolling)
- **Canvas Width:** Dynamic - calculated as (Number of Stages × 600px) + padding
  - Example: 5 stages = 3000px width minimum
  - Add 200px padding (100px left + 100px right) = 3200px total
- **Canvas Height:** Dynamic - based on number of swimlanes and content
  - Minimum per swimlane: 300px
  - Example: 3 swimlanes = 900px + 200px padding = 1100px total
- **Stage Column Width:** Max 600px (fixed)
- **Stage Spacing:** 40px horizontal gap between stages
- **Swimlane Height:** Auto (min 300px, expands with content)
- **Swimlane Spacing:** 20px vertical gap between lanes

#### Layout Direction Rules
```
Direction: HORIZONTAL
Flow: LEFT → RIGHT
Stages: [1] [2] [3] [4] [5] ...
Arrangement: Inline (not stacked)
```

#### Grid System
- **Horizontal Grid:** 600px stage columns + 40px gutters
- **Vertical Grid:** Swimlanes stacked with 20px gaps
- **Internal Padding:** 24px inside each stage column
- **Content Alignment:** Top-aligned within each swimlane

#### Typography
- **Stage Titles:** [Font family, 24px, bold]
- **Section Headers:** [Font family, 18px, semibold]
- **Body Text:** [Font family, 14px, regular]
- **Pain Point Labels:** [Font family, 12px, medium]

#### Spacing Scale
- **Stage Column Padding:** 24px
- **Element Spacing (vertical):** 16px between sections
- **Icon Spacing:** 8px from text
- **Swimlane Margin:** 20px vertical

#### Export Requirements
- **Primary Format:** Figma file (.fig)
- **Presentation Format:** PDF (landscape orientation)
- **Image Export:** PNG (2x resolution for high-DPI displays)
- **Interactive Prototype:** Figma prototype link with horizontal scroll enabled
